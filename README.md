# 타입스크립트 + Express 예제 앱

타입스크립트 + Express 예제 앱입니다.

## 사용된 기술

- ESLint
- webpack
- Docker
- vscode Debugger
- nodemon

## 실행

```
# 실행
> npm run dev
# 디버그 실행
> npm run dev:debug
```

## 빌드

```
# webpack 빌드
> npm run build
# Docker 빌드
> npm run docker-build
```

## 배포

```
# Docker 배포
> npm run docker-run
```
